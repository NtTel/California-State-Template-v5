GETTING STARTED

How do i get started?

To get started you need to download our State Template from GitHub. Also, you can refer to our Getting Started page

What what essential files do i need for clean State Tempale template?

  1. Core css and colorscheme files from /css folder (see our update page CSS)
  2. Core js files and other js libraries from /js folder (see our update page)
  3. Fonts files from /fonts folder
  4. Structure includes in /ssi folder (see our structure page)

What do I need to make server side includes (SSI) to work?

SSI must be set up on your server for them to work. You are unlikely to have SSI enabled if you do not have a CGI-BIN but if you do it is very likely that the server administrator will have enabled SSI.

INSTALLATION

How can I clone State Template from GitHub and bring it into my Visual Studio?

  1. Go to State Template GitHub repository page and click on "clone or download" button, then copy GitHub repository web URL
  2. Open your Visual Studio and go to the Team Explorer. Make sure your Visual Studio has GitHub extension.
  3. Click on the "Manage Connections" icon then in your GitHub connections section click on "clone" link. That will open up the "Open from GitHub" popup.
  4. Paste in State template GitHub repository web URL and click "Clone". That will clone the State Template from our GitHub repository to your local machine. Once it's cloned you can open the cloned website from your Visual Studio.

How do I set up local Docker container?

  1. Install Docker
  2. Download nginx Dockerfile and place this file in the same directory as your directory of content
  3. Run your docker image

Do I need to bring a boostrap library on top of State template?

No. State Template already uses the latest Bootsrap framework.

UPDATE

How can I update my existing website with the most currest State Tempalte version?

If your website is using a State Template that is perhaps couple of revisions back go to our update page where you will find all the necessary steps and details.

Is my old website will be compatible with State template latest vesrion?

Most likely. State Template was designed with backward compatibility in mind. The most current version that runs on Bootstrap 4 is mostly compatible with previous Boostrap 3 versions.
