Links

State Template hyperlink, or simply a link, is a reference to a webpage, or a document that the user can follow by clicking or tapping. To identify links inside of the paragraph text we are using blue color dotted underline style.

Regular link

This is a link example inside of the main body paragraph text.

Anchor link

An anchor hyperlink is a link bound to a portion of a document. To indicate anchor link you can add ca-gov-icon-anchor icon to it like this: 

<a href="#">anchor link <span class="ca-gov-icon-anchor link-icon" aria-hidden="true"></span></a>

External link

External Links are hyperlinks that point outside of your domain. To indicate external linkopens in new window you can add ca-gov-icon-external-link icon to it with disclaimer for the screen reader users that this hyperlink will open in the new window like this: 

<a href="https://ca.gov">anchor link <span class="ca-gov-icon-external-link link-icon" aria-hidden="true"></span><span class="sr-only">opens in new window</span></a>
