Dropdown Navigation

This page is using the Dropdown Navigation or Mininav.

The dropdown menu displays a simple dropdown for sub navigation elements. This style of menu can be set by including the .dropdown class in the <nav> element. like this <nav id="navigation" class="main-navigation dropdown"> .
