Bootstrap CSS and components or custom code

We recommend that you primarily use the template styles and components. However, if you have a need for more CSS classes or components, we recommend using what's available in Bootstrap first before developing custom code.

When you use custom code that is not part of the template, it causes you extra work when it’s time to go to the next version of the template or design system.
