Mobile Controls

Mobile Controls are the search and navigation toggle buttons that expand and collapse search and navigation sections in mobile view.

It includes:

  • Search toggle button.
  • Navigation toggle button.

When and how to use it

The Mobile Control one of the essential State Web Template structural components. Without it users would not be able to open site navigation or search bar in mobile view. It should be placed right below the site header.

By default, text identifier for search button is "Search" and text identifier for navigation button is "Menu".
