Settings Bar

Settings Bar is State Web Template's additional accessibility feature module that allows users to increase and decrease the font size, switch to a dyslexic font or toggle to a high contrast viewing mode.

It includes:

  • High contrast and default viewing mode buttons
  • Decrease and increase font buttons
  • Dyslexic font button
  • Font reset button
  • Close Settings Bar button

When and how to use it

The Settings Bar is activated by the settings toggle button that can be seen in the top right corner of the page. That button is part of the Utility Header component. The Settings Bar is hidden byt default and is placed right after Utility Header section.
