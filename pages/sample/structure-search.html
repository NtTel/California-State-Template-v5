Search

The Google Search component allows users to quickly find website content. Search component is optional feature but we have included it in State Web Template. Search component requires several parts to work together. In addition to the search bar form you need to set up the Google Search ID for your domain as well as the search results page for your website.

Active search

The active (featured) search is default search bar implementation on most State Web Template pages. This implementation makes search bar visible and available in the right side of the site header.

Inactive search

Another search implementation is the inactive (hidden) search bar. This implementation is used in mobile view and in pages that use Navigation with Icons site navigation type option. To activate the search bar users need to press search menu item first.

Usage

When and how to use it

Search component is placed into navigation-search container which goes after Mobile Controls. Search box usually displays to the right of the site name.

On mobile, the search box becomes a search button that opens a search box. It also displays to the right of the site name.

There are two kinds of search bar implementations in State Web Template.
