Usage

When and how to use it

Use social media icons within content (e.g. press release, blog post, etc). These icons should link to your official social media accounts.

Since these icons promote your social media accounts, make sure the content around them is clear. Use terms like "Follow us."

The available social media icons include:

  • Facebook
  • Twitter
  • Instagram
  • YouTube
  • GitHub
  • LinkedIn
  • Email

Browse our State Web Template icons.

How not to use it

Do not include more than one social media account per platform. Do not link to social media accounts that are not updated or monitored regularly.

This component is different from the social media links in the footer. The footer comes with social media options and does not use this component.
