Components

The following pages include component examples as well as documentation that describe the usage of each component. On any given page you will find a working example of the component, a description of the various classes used in the creation of the component, as well as the source code that you can copy and past from should you need to. The following pages help to serve as a styleguide as well as documentation.

Available components

Accordion

An expandable section of content that lets people choose what they want to see

Alert Banner

A space for high-priority information that can include an icon and links

Banner

The banner component is a full screen hero banner

Blockquotes

Blockquotes are used for quoting blocks of content from another source within your document

Breadcrumb

The breadcrumb indicate the current page's location within a navigational hierarchy

Buttons

We use the buttons for actions in forms, surveys, dialogs, and other call to action sections

Countdown

Countdown timer component is used to show countdown to a specific date or an event

Featured Narrative

The Featureed Narrative calls out an important bit of content within paragraphs

Featured Search

Featured search component is the active search box placed inside of the banner section

Form Styles

Form controls examples that have different styles and treatmets

Horizontal Rules

Horizontal rule is line element that is used to separate content

Lists

HTML lists allow web developers to group a set of related items in lists

Modal

Modals offer a lightweight, multi-purpose JavaScript popup thatâ€™s customizable and responsive

Number counter

Number counter is JavaScript based number animation to show some activity on the webpage

Parallax

Parallax scrolling is a feature where the background content image is moved at a different speed than the foreground content while scrolling

Progress Bar

Progress Bar provides up-to-date feedback on the progress of a workflow or action

Progress Blocks

Progress blocks is progress indicators that display progress through a sequence by breaking it up into multiple logical and numbered steps

Social Sharer

This module adds a group of buttons to your page that allows visitors to share the page via various social media services

Step List

The step list is a numbered list that has number for each item, lines connecting all the circled numbers, title and detail for each step

Tabs

Tabs lets you create content using list of links that are linked to a section elements with each its own content

Table

The table componenent is a structured set of data made up of rows and columns

Timeline

Timeline is a display of a list of events (or milestones) in chronological order

Additional Bootstrap CSS & Components

We strongly recommend using the documented State Template Content Modules and Components whenever possible; however, for more CSS classes and options as well as components we suggest utilizing what is available in Bootstrap first, before developing your own custom code.

NOTE: When using custom components and content modules that are not documented as part of the State Template, migration to future versions of the State Template may require additional work.
