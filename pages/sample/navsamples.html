Navigation Samples

Here are the State Web Template navigation types and variations:

Dropdown navigation

The dropdown menu displays a simple dropdown for sub navigation elements. Click on the image to experiense the Dropdown Navigation.

Flex megamenu

The Flex Megamenu displays dropdown panels for sub navigation. Click on the image to experiense the Flex Megamenu Navigation.

Full Width Navigation with Active Search

This wull width navigation layout uses featured search that is placed to the right of the website's logo.

Navigation with Icons

Navigation with Icons uses icons from CAGov font library inside of the first level navigation items. This navigation appears to the right of the site logo.

Page Navigation

Page navigation is a lists of anchor links to the level 2 headings (H2) of a webpage. It is placed at the top of the page on the left.

Side Navigation

Side navigation is a vertical navigation component that is placed at the top of the page on the left.

Single Level Navigation

This navigation type does not have a dropdown for sub navigation items.
