Page navigation

Page navigation is a lists and links to the level 2 headings (H2) of a webpage. It is placed at the top of the page on the left. People can select a link to go directly to H2 section. The links are usually introduced by the text "On this page". Page navigation makes it easy to see what content is available.

Note: The page navigation should not show if there are zero or one H2’s on the page.
